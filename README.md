# 🎲 今天吃什么

旨在帮助选择困难症解决人生难题

## 功能

- **加权随机**：健康分 1-5，分越高被选中概率越大（5 分项的选中概率是 1 分的 5 倍）
- **吃饭方式过滤**：外卖 / 到店 / 自己做，可多选
- **健康等级过滤**：⭐1 ~ ⭐5 可多选，精确控制想吃多健康的东西
- **穷鬼模式**：开关式，开启后仅保留 ≤ 15 元的选项
- **PWA 支持**：可添加到手机主屏幕，像原生 App 一样使用

## 快速开始

### 本地测试

```bash
python3 -m http.server 8080
# 打开 http://localhost:8080
```

> 注意：不能直接双击 HTML 文件打开，因为浏览器会阻止 `file://` 协议下的 fetch 请求。

### 部署到 GitHub Pages

1. 把本仓库 push 到 GitHub
2. Settings → Pages → Source 选 `main` 分支 → Save
3. 获得 `https://用户名.github.io/仓库名/` 链接
4. 手机浏览器打开 → 添加到主屏幕

## 自定义数据

编辑 `今天吃什么.md`，格式：

```
食物名 吃饭方式 价格 健康分(1-5)
```

- `#` 开头的行为注释，会被忽略
- 空行会被忽略
- 增删条目后 push 即可，GitHub Pages 自动更新

### 添加新食物后

如果新增了食物，需要在 `index.html` 的 `foodEmojis` 对象中添加对应的 emoji 图标，否则会显示默认图标 🍽️。

## 技术栈

纯 HTML + CSS + JS，零依赖，单文件 PWA。
